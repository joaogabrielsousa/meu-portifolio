Dev Front-End

O desenvolvedor front-end é o profissional responsável por criar a interface de sites e aplicativos, ou seja, tudo o que o usuário vê, clica e interage diretamente no navegador. Utilizando tecnologias como HTML, CSS e JavaScript, ele transforma layouts em páginas funcionais, garantindo que o design seja responsivo, ágil e acessível em diferentes dispositivos.

CSS

CSS (Cascading Style Sheets, ou Folhas de Estilo em Cascata) é a linguagem utilizada para definir o estilo visual de documentos HTML, controlando o layout, cores, fontes, espaçamentos e responsividade. Enquanto o HTML estrutura o conteúdo, o CSS "decora" e organiza a apresentação na web, permitindo que o design seja separado da estrutura para facilitar a manutenção.

JavaScript

JavaScript é uma linguagem de programação de alto nível, interpretada e dinâmica, essencial para criar interatividade e comportamento em páginas web, funcionando principalmente no navegador do usuário (cliente). Junto com HTML e CSS, forma a base da web moderna, permitindo funcionalidades como animações, atualizações de conteúdo, mapas e menus dinâmicos
